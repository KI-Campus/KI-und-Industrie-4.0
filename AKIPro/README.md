# Herzlich willkommen!

Hier finden Sie die praktischen Programmierübungen für den Kurs <i>Anwendung von KI in der Produktion</i>. Für jeden Anwendungsfall gibt es einen gleichnamigen Ordner, in dem Sie ein Jupyter Notebook finden, welches die entsprechende Programmierübung enthält. Das Notebook beinhaltet die Aufgaben, sowie erklärende Texte und Python-Code. Die Lösungen der einzelnen Aufgaben können direkt im Notebook geladen werden. 

Eine Anleitung für die notwendigen Installationen, um lokal auf Ihrem Rechner mit den Jupyter Notebooks zu arbeiten und die Übungsaufgaben zu lösen, finden Sie im ersten Kursmodul <i>Einführung</i> unter <i>Installation der Übungsumgebung</i>.

Viel Freude bei der Bearbeitung der Programmierübungen!
